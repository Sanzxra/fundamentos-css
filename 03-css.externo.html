Conhecendo a abordagem de CSS Externo

O CSS Externo consiste em usar uma folha de estilos externa.

É uma das práticas mais recomendadas em desenvolvimento web, pois ajuda a organizar o código, reutilizar estilos e manter o HTML limpo
